# GitDoc Start

## Project Overview

This repository is a React + Vite frontpage clone built with:
- `react` 19
- `vite` 4
- `tailwindcss` 4
- `swiper` for slides/carousels
- `react-icons`

The application is organized as a single-page app with reusable components in `src/components`.

## Running the App

Install dependencies:
```bash
npm install
```

Start development server:
```bash
npm run dev
```

Build for production:
```bash
npm run build
```

Preview built output:
```bash
npm run preview
```

## Key Files

- `package.json` - project metadata and scripts
- `vite.config.js` - Vite configuration
- `index.html` - app shell
- `src/main.jsx` - application bootstrap
- `src/App.jsx` - root component
- `src/index.css` - global styles

## Component Structure

`src/components` contains:
- `Navbar/` - top navigation
- `Hero/` - page hero section
- `About/` - about section
- `Courses/` - courses showcase
- `Learning/` - learning section
- `Assessments/` - assessments section
- `Awards/` - awards display
- `Companies/` - employer/company logos
- `Corporate/` - corporate information
- `Explore/` - exploration section
- `Footer/` - site footer
- `Learners/` - learner stories
- `Partners/` - partners section
- `Proud/` - pride or community section
- `Resources/` - resources section
- `Testimonials/` - testimonials section
